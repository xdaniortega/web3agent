# web3agent-sdk Documentation

Deploy autonomous AI agents on Arbitrum with LangChain, isolated wallets, and ERC-8004 identity.

## Quick start (3 steps)

1. `cp .env.example .env` — fill in your API keys
2. Fund the master wallet with Sepolia ETH
3. `npx tsx scripts/test-workflow.ts`
